## Installation locale

1. Clonez le repository
```bash
git clone https://github.com/gracemakeu26-lgtm/Makeu-Portfolio.git
cd Makeu-Portfolio
```

2. Démarrez le serveur PHP
```bash
php -S localhost:8000
```

3. Ouvrez votre navigateur à `http://localhost:8000`

## Déploiement sur Render

Ce projet est configuré pour un déploiement facile sur Render :

1. Connectez-vous à [Render](https://render.com)
2. Créez un nouveau Web Service
3. Connectez votre repository GitHub
4. Render lira automatiquement `Procfile` et `render.yaml`
5. Cliquez sur "Create Web Service"
